# maven-project-advanced-class
This is a maven project from MuleSoft's advanced development class

## How to run the project
1. Add the remote repository: `git remote add origin https://github.com/JakeiCIMS/maven-project-advanced-class`
2. Enter the repo: `cd maven-­-project`
3. Set your MULE_HOME env variable: `export MULE_HOME={locationOfMuleInstall}`
4. Package and deploy: `mvn install`
